# ✨ AI Editorial Calendar

A beautiful, modern editorial calendar with a premium dark UI built with Next.js, Framer Motion, and Tailwind CSS.

## 🚀 Live Demo

**[View Live Demo](https://editorial-calendar-kgktx8oon-himanshkataria657-8702s-projects.vercel.app/)**

## ✨ Features

- 🌑 **Dark Premium UI** - Glassmorphism effects with gradient backgrounds
- 📅 **Month Navigation** - Browse through different months easily
- 📝 **Smart Notes System** - Add, edit, and delete notes for any day
- 💾 **Auto-save** - All notes saved automatically to localStorage
- 🔍 **Search Functionality** - Find notes quickly with real-time search
- 📊 **Statistics Dashboard** - Track total notes and monthly activity
- ✨ **Smooth Animations** - Beautiful Framer Motion animations throughout
- 🎨 **Gradient Glow Effects** - Premium hover effects on calendar days
- 📱 **Fully Responsive** - Works perfectly on all devices
- ⚡ **Fast & Clean** - Optimized performance with Next.js

## 🛠️ Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **date-fns** - Date utilities

## 🏃 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 📦 Installation

```bash
npx create-next-app@latest editorial-calendar
cd editorial-calendar
npm install framer-motion date-fns
```

## 🎨 Features Breakdown

### Calendar View
- Full month calendar with week layout
- Today indicator with pulsing animation
- Previous/Next month navigation

- Current month notes
- Search results counter
